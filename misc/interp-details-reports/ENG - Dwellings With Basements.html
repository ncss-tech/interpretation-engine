This report locates 1,249 items (data elements using either physical or logical name) in the parent property and in up to 16 properties called in derive statements. The data elements are in the area, area type, chorizon, chorizon aashto, chorizon consistence, chorizon desgn suffix, chorizon fragments, chorizon human artifacts, chorizon pores, chorizon structure, chorizon structure group, chorizon texture, chorizon texture group, chorizon texture modifier, chorizon unified, component, component canopy cover, component crop yield, component diagnostic features, component erosion accelerated, component existing plants, component forest prod, component forest prod other, component geomorph desc, component hydric criteria, component month, component parent material, component parent material grp, component potential windbreak, component restrictions, component soil moisture, component soil temperature, component surface fragments, component surface morph gc, component surface morph hpp, component surface morph mr, component surface morph ss, component tax fam mineralogy, component tax fam other, component tax moisture class, component usfs ground cover, component usfs interpretation, correlation, data mapunit, geomorph feature, geomorph feature type, legend, legend mapunit, and mapunit tables. It does not search for data elements ending in "iid", "iidref", rvindicators, sequence numbers, or object and record when last updated items.

Main properties occur first and are in grey. 1st-level derive properties are in yellow, 2nd-level in green, 3rd-level in blue, 4th-level in purple, and 5th-level in orange.

The report works by isolating the word DERIVE and a line feed at the end of the derive statement. It must do this for every derive statement and this report can process up to 16 of them. It then isolates the NASIS site and property name using the first double quote, the colon, and the last double quote. In cases where the NASIS site is omitted, it is assigned from the parent property. With carefully constructed case statements, the code can accommodate either scenario. Derive statements preceeded by either a space or a hashtag are not included.

This report has no method to distinguish between data elements being used and those that have been hashed out with #. This unresolved issue remains significant.
